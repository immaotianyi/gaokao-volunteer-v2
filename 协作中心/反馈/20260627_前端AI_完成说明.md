### 2. RadarCard.vue — 热度徽章

**渲染位置**：`card-head` 的 `card-info` 内，院校名 + school_type 之后。
- 选择此位置是为了与右上角的「重点推荐」徽章（tier-s 时显示）不冲突
- 仅当 `item.heat_level` 存在时渲染（v-if="heatLevel"）

**hover title**（鼠标悬浮显示明细）：
```
累计 {heat_view_count} 人查看 · 今日 {heat_today_view} 人 · {heat_watcher_count} 人在盯
```
三个数值均经 `Math.max(0, n ?? 0)` 兜底，防 #5 后端修复前的负数。

**配色（秉烛研卷主题：墨色底 + 烛光琥珀）**：
| heat_level | 文案示例 | 配色 | 特效 |
|------------|----------|------|------|
| cold | 少人关注 | 灰墨色 rgba(148,163,184,0.1) #94a3b8 | 无 |
| normal | 正常关注 | 默认琥珀 rgba(232,185,116,0.1) #f4d8a8 | 无 |
| hot | 热门机会 | 加亮琥珀 rgba(251,191,36,0.15) #fde68a | 无 |
| viral | 爆款机会 | 强调琥珀 rgba(250,204,21,0.18) #facc15 | 呼吸光效 2.4s |

viral 呼吸光效 keyframes：
```css

  0%, 100% { box-shadow: 0 0 6px rgba(250, 204, 21, 0.25); }
  50%      { box-shadow: 0 0 14px rgba(250, 204, 21, 0.55); }
}
```
