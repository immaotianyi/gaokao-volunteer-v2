§ 1. 创意介绍

想解决什么问题： 每年高考志愿填报季，超过 30% 的退档事件源于考生忽略招生章程中的隐性规则—— 体检限制（色盲/色弱报临床直接被退档）、单科门槛（数学低于100分报数据科学被退档）、语种要求（日语考生报了仅限英语专业）、选科不符。 家长和考生面对 135+ 所大学 的数万字章程无从下手，信息过载、认知门槛极高。

为什么会想到做这个： 身边每年都有亲戚朋友家孩子高考完对着志愿填报指南发愁——不是不知道想报什么，而是不知道报什么不会"踩雷"。 一份体检报告 + 一份志愿草表 + 数万字章程条款，全靠人工逐条核对，非常容易遗漏致命风险。我们想用 AI 把这件事变成"粘贴草表 → 30秒出报告"。

产品形态： Web 端单页应用，核心技术栈 Vue 3 + FastAPI + DeepSeek 大模型 + RAG 结构化知识库 + PostgreSQL。无需下载 App，打开浏览器即可使用。

§ 2. 目标用户及痛点

👨‍🎓
高考应届考生
出分后需要在 3-5 天内完成志愿填报，面对 45 个志愿格无从下手。大多数家庭没有专业咨询师资源，全靠"问亲戚+刷贴吧"。
👨‍👩‍👧
考生家长
想帮孩子但不懂新高考规则（选科组合 / 专业组投档 / 平行志愿），焦虑且容易轻信短视频平台的碎片化信息。

核心痛点

⚠
章程规则碎片化135+ 所大学章程分散在各校官网，格式不统一，查阅一所需 5-10 分钟，全部核对需数小时。
⛔
隐性规则遗漏体检限制 / 单科门槛等条款隐蔽在章程正文中，人工逐条核对极易遗漏，一旦触发 = 退档。
🔍
捡漏机会不可见新增专业 / 扩招专业 / 新校区首招等低竞争机会靠人工搜索几乎不可能全面覆盖。
💰
付费咨询门槛高专业志愿填报机构收费 3000-8000 元，普通家庭难以承担，且服务质量参差不齐。

§ 核心功能一览

🛡

志愿探雷器

粘贴志愿草表，AI 逐条审查每所大学的招生章程。四维交叉比对（体检+单科+语种+选科），DANGER/WARNING/PASS 三级定级，结果可溯源到具体章程条款。

📡

捡漏雷达

全网扫描新增专业、扩招计划、纯净专业组，六维评分系统 0-100 分锁定捡漏机遇。含估值模型为新增专业预估分数线。

🤖

AI 志愿顾问

基于章程知识库 + 雪峰志愿方法论 + 联网搜索，多轮对话深度答疑。你的分数、位次、体检标记全部注入上下文，回答精准可落地。

§ 3. 价值与意义

Social Value · 教育公平
  • 降低高考志愿填报的信息不对称 —— 让没有专业咨询师资源的普通家庭也能获得高质量志愿审查
  • 用 AI 将数万字章程结构化，使"合规审查"从数小时缩至 30 秒
  • 探雷器 + AI 顾问永久免费，捡漏雷达前 5 条免费，确保教育公平
  • 隐私优先：档案仅存本地浏览器，不向第三方分享
Efficiency · 效率革命
  • 四维交叉审查引擎（体检 + 单科 + 语种 + 选科）—— 一次性覆盖所有退档维度
  • 六维捡漏评分系统（新增专业 + 扩招 + 新校区 + 纯净组 + 选科稀缺 + 分数匹配）
  • SSE 流式推送，用户实时观察 AI 推理过程，每一步透明可溯源
  • V3 规则引擎 + V4 AI Agent 双模式，离线规则引擎保障零 API 成本兜底
